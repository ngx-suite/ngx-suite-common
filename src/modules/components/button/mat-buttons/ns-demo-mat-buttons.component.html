Custom directives are used to style standard MatButton. On top of standard mat-button, mat-stroked-button mat-flat-button and mat-icon-button directives, there are custom directives that can be used in combination with the standard mat-button directives:

  • [mat-button][nsButtonSize],
  • [mat-button][nsButtonStyle],
  • [mat-button][color].

In addition, there is also custom directive [nsFlatButton], [nsStrokedButton] that can be used instead of generic [nsButtonStyle] to defined the particular style:

  • [mat-button][nsFlatButton]
  • [mat-button][nsStrokedButton]
